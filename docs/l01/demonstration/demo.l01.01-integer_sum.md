# Demonstration #01: Integer Sum

## Problem

Given two integers, `a` and `b`, find the sum of `a` and `b`.

## Example

Input:

    a = 5
    b = 7

Output:

    a + b = 12

## Pseudocode

### Aproach #01

```pseudocode
INICIO
    ED: numero1, numero2, soma INTEIRO
    
    LER (numero1, numero2)
    
    soma <- numero1 + numero2
    
    ESCREVER (soma)
FIM
```

### Aproach #02

```pseudocode
INICIO
    ED:
        numero1, numero2, soma INTEIRO

    LER (numero1)
    LER (numero2)
    
    ESCREVER (numero1 + numero2)
FIM
```
