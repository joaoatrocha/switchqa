# Demonstration #05: Display "Positive" or "Negative" or "Zero"

## Problem

Display “Positivo” or “Negativo” or “Zero” according to the given number.

## Example

Input:

    number = 1

Output:

    Positivo

Input:

    number = -1

Output:

    Negativo

Input:

    number = 0

Output:

    Zero

## Pseudocode

### Aproach

```pseudocode
INICIO
    ED: numero INTEIRO 
    LER (numero)

    SE (numero = 0) ENTÃO
        ESCREVER (“Zero”) 
    SENÃO
        SE (numero > 0) ENTÃO 
            ESCREVER (“Positivo”)
        SENÃO
            ESCREVER (“Negativo”)
        FIMSE 
    FIMSE
FIM
```
